# 🎯 InterviewIQ.AI – AI-Powered Mock Interview Platform

InterviewIQ.AI is a full-stack web application that helps users prepare for job interviews using Artificial Intelligence. The platform generates personalized interview questions, supports voice-based answers, evaluates responses, and provides detailed feedback with scores and improvement suggestions.

---

## 🚀 Features

- 🔐 User Authentication (Email/Password + Google Login)
- 👤 Profile Setup with Role and Experience
- 📄 Resume Upload and Analysis
- 🤖 AI-Generated Interview Questions using OpenRouter API
- 🎤 Voice-to-Text Answer Input
- 📝 Manual Answer Submission
- 📊 AI-Based Feedback and Scoring
- 📈 Performance Analytics and Interview History
- 🛠️ Admin Dashboard for User and Question Management
- 💳 Razorpay Payment Integration (Optional)
